## 🏛️ 版权申明

本工程是 `金庸群侠传X v1.1.0.6` 版本通过逆向工程获得，仅用于给爱好游戏开发和喜欢金庸类游戏的开发者做为学习参考，仅用于学习和研究目的，无任何商业用途，游戏内部许多素材均来自于互联网，**禁止使用本游戏中任何元素应用到任何商业行为中**，若有侵犯您的权益，请与我们联系。

## 🛠️ 开发环境

- 🥋 代码开发：[Rider](https://www.jetbrains.com/rider/)
- 🎮 游戏引擎：[Unity](https://unity.com/)

## 🎯 实现功能

- [x] 使用最新的 `LTS` 版本的 `Unity` 进行游戏开发
- [x] 使用 `xLua` 替换原有的 `uLua` 版本，并对原有的 `lua` 进行适配

## 🍃 特别鸣谢

- [xLua](https://github.com/Tencent/xLua): `C#` 下 `Lua` 编程支持
- [UnityTextToTextMeshProUpgradeTool](https://github.com/yasirkula/UnityTextToTextMeshProUpgradeTool.git): `Unity Text` 到 `TMPro` 转换工具
- [UnitySimpleFileBrowser](https://github.com/yasirkula/UnitySimpleFileBrowser.git): 文件系统浏览器组件
